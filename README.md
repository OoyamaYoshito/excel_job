# excel_job
エクセルファイルから学生のアンケート情報を出力するプログラムです

# 環境
- Python 3.6.7
- pandas 0.23.4
- xlrd 1.1.0

# インストール方法
[PythonインストールWindows](https://www.python.jp/install/windows/install_py3.html)

[PythonインストールMac](https://qiita.com/ms-rock/items/6e4498a5963f3d9c4a67)

`$ pip install pandas`

`$ pip3 install xlrd`
